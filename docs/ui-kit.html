BAPLCP 首頁 UI Kit

這份文件整理首頁目前使用的視覺規則、字級、顏色與抽屜式 dialog 樣式。日後新增首頁元件時，優先沿用這裡的 token、尺寸與狀態命名。

文字敘述規則

  • 文案保持短句，首頁標題使用「功能區」或「任務」語氣，例如「球局報名區」。
  • 副標用一行補充使用情境，例如「最新臨打報名及季打請假」。不要在卡片內寫操作教學。
  • 主要行動按鈕使用明確動詞，例如「立即前往」、「建立新球局」。
  • 未完成功能使用 is-pending 狀態，透明度 40%，並停用點擊。

視覺規則

  • 頁面背景固定為淺紫藍，不使用深色模式；首屏載入也要維持 light scheme。
  • 手機容器最大寬度 390px，桌機置中顯示；手機 viewport 下鋪滿全寬。
  • 主要品牌區使用紫藍垂直漸層，輔助行動使用奶油橘漸層。
  • 卡片圓角 16px，主要內容面板圓角 20px，抽屜底部按鈕圓角 12px。
  • 圖示入口使用 76px 固定區塊與圓形底，避免載入圖示時版面跳動。
  • 抽屜 dialog 從右側滑入，寬度為 min(329px, calc(100% - 61px))，遮罩為 40% 黑。
